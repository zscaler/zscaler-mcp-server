# Z-Insights

~17 analytics tools across web traffic, threat trends, CASB, shadow IT, and IoT.

## Tool families

- **Web traffic** — By location, by protocol, no-grouping rollups
- **Threats** — Super categories, classes
- **Cyber incidents** — By category, by location, daily trends, by threat + app
- **Firewall** — By action, by location, network services
- **CASB** — SaaS application usage report
- **Shadow IT** — Discovered apps with risk scores, summary statistics
- **IoT** — Device stats and classifications

## Toolset

Single toolset: `zins`.
