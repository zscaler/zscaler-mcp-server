# ZCC — Zscaler Client Connector

Small read-only service exposing device enrollment, forwarding profiles, and trusted networks.

## Tools

- `zcc_list_devices` — ZCC device enrollment information from the Client Connector Portal
- `zcc_list_trusted_networks` — Trusted networks by company ID
- `zcc_list_forwarding_profiles` — Forwarding profiles by company ID

## Toolset

Single toolset: `zcc`.
