# EASM — External Attack Surface Management

~7 read-only tools for managing internet-facing assets and lookalike domains.

## Tools

- `zeasm_list_organizations` — List all EASM organizations
- `zeasm_list_findings` — List findings for internet-facing assets
- `zeasm_get_finding_details` — Detailed finding information
- `zeasm_get_finding_evidence` — Scan evidence for a finding
- `zeasm_get_finding_scan_output` — Complete scan output
- `zeasm_list_lookalike_domains` — Lookalike domains detected for an organization
- `zeasm_get_lookalike_domain` — Details for a specific lookalike domain

## Toolset

Single toolset: `zeasm`.
