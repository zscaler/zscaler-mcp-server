Client API¶

The Zscaler client wrapper and authentication handling.

get_zscaler_client(client_id=None, client_secret=None, customer_id=None, vanity_domain=None, private_key=None, username=None, password=None, api_key=None, key_id=None, key_secret=None, secret_key=None, cloud=None, service=None, use_legacy=False)[source]¶

Client Configuration¶

The client supports both OneAPI and Legacy API authentication methods.

Usage Examples¶

OneAPI Client¶

from zscaler_mcp.client import get_zscaler_client

# Using environment variables
client = get_zscaler_client()

# Using explicit parameters
client = get_zscaler_client(
    client_id="your_client_id",
    client_secret="your_client_secret",
    customer_id="your_customer_id",
    vanity_domain="your_vanity_domain"
)

Legacy Client¶

from zscaler_mcp.client import get_zscaler_client

# ZIA Legacy
client = get_zscaler_client(
    use_legacy=True,
    service="zia",
    username="your_username",
    password="your_password",
    api_key="your_api_key",
    cloud="your_cloud"
)

# ZPA Legacy
client = get_zscaler_client(
    use_legacy=True,
    service="zpa",
    client_id="your_client_id",
    client_secret="your_client_secret",
    customer_id="your_customer_id",
    cloud="your_cloud"
)
